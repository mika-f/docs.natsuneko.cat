## インストール

1. [BOOTH](https://natsuneko-vrc.booth.pm/items/3853084) からインストールアーカイブをダウンロードします。
2. アーカイブを解凍し、 `TextMeshCreator.unitypackage` をダブルクリックし、インストールします。
3. Unity のメニューから `Windows > Natsuneko Laboratory / TextMesh Creator Installer` をクリックし、必要なパッケージをインストールします。
4. Unity のメニューから `Windows > Natsuneko Laboratory / TextMesh Creator` をクリックし、TextMesh Creator ウィンドウを開きます。

## 更新

1. [BOOTH](https://natsuneko-vrc.booth.pm/items/3853084) から最新のインストールアーカイブをダウンロードします。
2. アーカイブを解凍し、 `TextMeshCreator.unitypackage` をダブルクリックし、インストールします。

## 削除

`NatsunekoLaboratory/TextMeshCreator` フォルダを Unity プロジェクトから削除します。
