title: パラメーター (Unity 版)

---

TextMesh Creator には、テキストメッシュをカスタマイズするための以下のパラメーターがあります。

| Property       | Required | Type     | Description                                              |
| -------------- | :------: | -------- | -------------------------------------------------------- |
| フォント       |   Yes    | `string` | 使用するフォント (ttf もしくは otf)                      |
| テキスト       |   Yes    | `string` | 表示する文字列                                           |
| フォントサイズ |   Yes    | `float`  | 使用するフォントサイズ                                   |
| 押し出しの深さ |   Yes    | `float`  | 文字メッシュの押し出しの深さ                             |
| スケール       |   Yes    | `float`  | 文字メッシュのスケール                                   |
| カーブ分割数   |   Yes    | `int`    | 文字メッシュのカーブ分割数、大きいほど滑らかになります。 |
| マテリアル     |   Yes    | `string` | 使用するマテリアル                                       |
| パーツ毎に分割 |    No    | `bool`   | "Vertex" で分離                                          |
| 保存場所       |   Yes    | `string` | テキストメッシュをエクスポートするディレクトリ           |
| 開始番号       |    No    | `int`    | インクリメントを開始する数                               |
| ユニーク       |    No    | `bool`   | 出力ファイルを一意にフィルタリング                       |

## "パーツ毎に分割" オプションについて

"パーツ毎に分割" プロパティは、テキストメッシュを "パーツ毎に分割" で分割するために使用されます。
"パーツ毎に分割" とは、テキストメッシュが指定された **Vertex** で分割されることを意味します。
たとえば、"あ" を以下のテキストメッシュに適用すると、テキストメッシュは「あ」の画数毎に分割されます。アルファベットなど単純な要素は分割されません。
