Astro Notes - Locky Cam は、 VRChat におけるリアルタイムライブでのカメラワークを制御するためのスクリプトです。
範囲内の特定のユーザーに対し、動きに追従するカメラワークを提供。
また、カメラ毎に UI から細かい位置や拡大率などの調整を行うことができ、より臨場感のあるライブ演出をサポートします。
