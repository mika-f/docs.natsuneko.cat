---
title: Author ID について
---

Epiclese の Author ID は、ワールドの作者を識別するための一意の文字列です。
プラットフォーム毎に ID の形式は異なり、それぞれ以下の形式で表されます。

## VRChat

VRChat における Author ID は、`usr_` で始まる英数字の文字列です。

例：

- https://vrchat.com/home/user/usr_d37acd76-341a-46af-9b0e-d5b0ffa2181d
- usr_d37acd76-341a-46af-9b0e-d5b0ffa2181d

確認するには、ワールドのページから、作者ページへ移動し、URL の末尾の部分を参照してください。

## Resonite

Resonite における Author ID は、`U-` もしくは `G-` で始まる英数字の文字列です。

例：

- U-esnya
- G-Team-VibeZ

確認するには、 https://go.resonite.com/world からワールドのリソース URL の初めの部分を参照してください。

例：

- `resrec:///U-Akataru1/R-6718e287-1097-476e-8ee1-6eb03bfd9641` であれば、 `U-Akataru1` が Author ID です。

## Cluster

Cluster における Author ID は、ユーザーページの末尾にある英数字の文字列です。

例：

- https://cluster.mu/u/ichiganryuta
- ichiganryuta

確認するには、ワールドのページから、作者ページへ移動し、URL の末尾の部分を参照してください。
