---
title: World ID について
---

Epiclese の World ID は、ワールドを識別するための一意の文字列です。
プラットフォーム毎に ID の形式は異なり、それぞれ以下の形式で表されます。

## VRChat

VRChat における World ID は、`wrld_` で始まる英数字の文字列です。

例：

- https://vrchat.com/home/world/wrld_a5e9ec13-36b1-4e63-ae0c-dab9023401f9/info
- https://vrchat.com/home/world/wrld_a5e9ec13-36b1-4e63-ae0c-dab9023401f9
- wrld_a5e9ec13-36b1-4e63-ae0c-dab9023401f9

確認するには、ワールドのページ URL の末尾の部分を参照してください。**インスタンスの ID ではありません。**

## Resonite

Resonite における World ID は、`R-` で始まる英数字の文字列です。

例：

- https://go.resonite.com/world/U-esnya/R-3acf68b3-b78b-4a8c-96ef-97a2f744602f
- U-esnya/R-3acf68b3-b78b-4a8c-96ef-97a2f744602f
- R-3acf68b3-b78b-4a8c-96ef-97a2f744602f

確認するには、 https://go.resonite.com/world からワールドのリソース URL の末尾の部分を参照してください。

## cluster

Cluster における World ID は、 UUID 形式の英数字の文字列です。

例：

- https://cluster.mu/w/20d14291-1d6b-4ad9-9f4e-fe8b87dd6f0c
- 20d14291-1d6b-4ad9-9f4e-fe8b87dd6f0c

確認するには、ワールドのページ URL の末尾の部分を参照してください。
